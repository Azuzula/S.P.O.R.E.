# S.P.O.R.E.
### Shared Posts Over Remote Ecosystems

SPORE adds a social layer on top of existing websites.

No new platform.  
No new feed.  
No walled garden.

Just conversations growing where the web already lives.

## ğŸ§¬ What is SPORE?

SPORE is a browser extension that allows users to create and read discussions directly on any webpage.

Instead of moving people to another social network, SPORE attaches conversations to the page itself.

Think of it as:

- a hidden discussion layer
- a decentralized comment system
- an experiment in social infrastructure
